**IP** (Internet Protocol – досл. «межсетевой протокол»), адресный протокол, который отвечает за передачу информации в сети, обеспечивая доставку [пакетов](https://tcinet.ru/press-centre/glossary/article.php?ELEMENT_ID=4931) данных между любыми узлами сети через произвольное число промежуточных узлов ([маршрутизаторов](https://tcinet.ru/press-centre/glossary/article.php?ELEMENT_ID=4933)). Неотъемлемой частью протокола является адресация сети: для точной передачи информации необходимо задать координаты получателя ([IP-адрес](https://tcinet.ru/press-centre/glossary/article.php?ELEMENT_ID=4924)). Именно IP стал тем протоколом, который объединил отдельные компьютерные сети во всемирную сеть [Интернет](https://tcinet.ru/press-centre/glossary/article.php?ELEMENT_ID=4920 "Интернет"). Однако протокол IP не гарантирует надежной доставки данных до адресата. В частности, пакеты информации могут быть потеряны в процессе передачи, присланы не в том порядке, в котором были отправлены, а также могут продублироваться (приходят две копии одного пакета) или оказаться поврежденными (обычно поврежденные пакеты уничтожаются). То есть использования одного лишь IP-протокола может быть недостаточно для корректной передачи данных. Гарантию безошибочной доставки пакетов дает, например, протокол [TCP](https://tcinet.ru/press-centre/glossary/article.php?ELEMENT_ID=4929 "TCP"), который использует IP в качестве транспорта.
