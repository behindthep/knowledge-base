**Клоакинг** (от англ. cloak — мантия, маска, прикрытие) – прием «черной» поисковой оптимизации, заключающийся в том, что информация, выдаваемая пользователю и поисковым роботам на одной и той же странице, различается. Хорошо оформленные, легко читаемые, удобные для посетителей [сайта](https://tcinet.ru/press-centre/glossary/article.php?ELEMENT_ID=5200) страницы могут проигрывать в [выдаче](https://tcinet.ru/press-centre/glossary/article.php?ELEMENT_ID=5153) по [поисковым запросам](https://tcinet.ru/press-centre/glossary/article.php?ELEMENT_ID=5205) (хотя бывают исключения), а оптимизированные страницы, специально насыщенные ключевыми словами, могут быть неудобны для пользователей. По этим причинам был создан метод клоакинга, для реализации которого создаются два варианта страницы: один для поисковых роботов, другой — для посетителей. Пользовательская страница оформляется произвольным образом, без каких-либо ограничений, связанных с поисковой оптимизацией. Страница, предназначенная для [краулеров](https://tcinet.ru/press-centre/glossary/article.php?ELEMENT_ID=5214), наполняется и оформляется в соответствии с требованиями оптимизации, для чего нужно иметь знания о поисковых системах. Для различения роботов и посетителей могут проверяться их [IP-адреса и user-agent'ы](https://tcinet.ru/press-centre/glossary/article.php?ELEMENT_ID=4924).
