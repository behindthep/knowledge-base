**URL** (англ. Uniform Resource Locator — единый указатель ресурса ) — стандартизированный способ записи адреса ресурса в сети [Интернет](https://tcinet.ru/press-centre/glossary/article.php?ELEMENT_ID=4920), был изобретен в 1990 году и изначально предназначался для обозначения мест расположения ресурсов (чаще всего файлов) во Всемирной паутине. Сейчас URL применяется для обозначения адресов почти всех ресурсов Интернета. Стандарт URL регулируется организацией [IETF](https://tcinet.ru/press-centre/glossary/article.php?ELEMENT_ID=4866) и ее подразделениями. Структура URL выглядит следующим образом: <схема>:[//[<логин>:<[пароль](https://tcinet.ru/press-centre/glossary/article.php?ELEMENT_ID=5177)>@]<[хост](https://tcinet.ru/press-centre/glossary/article.php?ELEMENT_ID=4923)>[:<[порт](https://tcinet.ru/press-centre/glossary/article.php?ELEMENT_ID=5193)>]][/][?<параметры>][#<якорь>]

Под схемой подразумевается сетевой протокол, хост представляет собой полностью прописанное [доменное имя](https://tcinet.ru/press-centre/glossary/article.php?ELEMENT_ID=7503) хоста в системе [DNS](https://tcinet.ru/press-centre/glossary/article.php?ELEMENT_ID=4864) или [IP-адрес](https://tcinet.ru/press-centre/glossary/article.php?ELEMENT_ID=4924) хоста, URL-путь - уточняющая информация о месте нахождения ресурса, а якорем может быть, например, заголовок внутри документа.
