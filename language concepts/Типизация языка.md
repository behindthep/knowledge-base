**Статическая** (у Java): Тип переменной установленный при определении не может измениться во время исполнения. Компилятор проверяет типы и синтаксические ошибки без запуска.

int age = 44;

age = "Not so old"; // Error

**Динамическая**: Тип может меняться по ходу исполнения программы.

age = "Not so old"; // OK

**Слабая/Не строгая**: Язык сам приводит типы переменных.

System.out.println("12" + 13); // 1213  
System.out.println(12 + "13"); // 25

**Сильная/Cтрогая** (у Java): Ручное приведение.

System.out.println("12" + Integer.toString(13)); // 1213
